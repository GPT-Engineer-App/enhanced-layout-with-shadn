## Setup

```sh
git clone https://github.com/GPT-Engineer-App/enhanced-layout-with-shadn.git
cd enhanced-layout-with-shadn
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
